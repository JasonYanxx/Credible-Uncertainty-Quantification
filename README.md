# TIM Credibility Code

This repository is a minimal public package for reproducing the main results of the TIM credibility study.

It includes:

- the core method implementation in `src/`
- the paper's simulation scripts in `experiments/simulation/`
- the paper's real-data scripts in `experiments/real_data/`
- a small set of public-facing reproducibility notes

## Data

The real-data pipeline depends on STARloc data that is not redistributed here.

Expected local filenames:

- `data/external/starloc_data_grid_s3_uwb.csv`
- `data/external/uwb_markers_v2.csv`

Original source:

- [utiasASRL/starloc](https://github.com/utiasASRL/starloc)

See [data/external/README.md](data/external/README.md) for details.

## Quick Start

1. Install dependencies:

```bash
pip install -r requirements.txt
```

1. Reproduce the paper in manuscript order:

- follow [REPRODUCIBILITY.md](REPRODUCIBILITY.md)

1. Check the paper-to-script mapping:

- see [docs/paper-traceability.md](docs/paper-traceability.md)

## Paper

- [arXiv:2509.03311](https://arxiv.org/abs/2509.03311)

## Repository Layout

- `src/`: reusable algorithm and helper modules
- `experiments/simulation/`: synthetic experiments used in the paper
- `experiments/real_data/`: STARloc/UWB workflow used in the paper
- `data/external/`: placeholder location for restricted external data
- `docs/`: public-facing paper mapping and reproduction notes
- `results/`: local output directory created during reruns

## Notes

- `Table I` and `Fig. 2` in the paper are not generated by scripts in this package.
- The remaining figures and tables are mapped to runnable scripts.
- Minor numerical differences can occur in Monte Carlo experiments due to randomness, but the qualitative conclusions should remain the same.
